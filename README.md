## ✨ Features

- **Advanced Blacklist System**
- Add a member to Blacklist
- remove member from Blacklist
- set Blacklist Role


## Commands
```
b-add اضافة عضو للبلاك ليست
b-remove ازالة عضو من البلاك ليست
b-role رول البلاك ليست

```
